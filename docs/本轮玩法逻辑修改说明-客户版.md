# 明日之问｜本轮玩法逻辑修改说明（客户版）

> 日期：2026-05-22
> 用途：向客户说明本轮需求已落地后的玩法结果和数值口径。
> 说明方式：本文只解释玩家能看到、能理解、能游玩的规则结果，不展开技术实现细节。

## 1. 本轮交付结论

本轮已经把市场、工厂、财政、建交和封锁相关逻辑统一到当前确认口径。玩家现在看到的是一套更直接的工业化经营循环：

1. 用工厂预算购买材料、扩建工厂或升级工厂。
2. 用材料生产统一商品。
3. 把商品卖到国内或海外。
4. 销售收入回流到民间购买力、工厂预算和政府财政。
5. 再用新的资源继续扩大生产、研究科技、争夺海外容量和海上控制。

| 客户需求 | 当前版本结果 |
| --- | --- |
| 国内市场按需求和购买力计算 | 已改为动态价格：民间购买力越高、需求越高，国内价格越有支撑；一次投放过多会降价，但不会禁止出售。 |
| 海外市场固定价格 | 已改为区域固定价：欧洲最高，非洲最低；政策不再提高海外单价。 |
| 海外市场不要建交门槛 | 已删除建交前置；海外默认开放，主要受容量和封锁影响。 |
| 政府侧只保留一个海外扩容政策 | 已只保留“贸易促进”，作用是提高海外容量，不提高海外价格。 |
| 市场竞争不加价格 | 已改为竞争成功只增加该区域额外可售容量。 |
| 海军演练不要作为海外扩容入口 | 已从当前玩家动作中移除。 |
| 工厂用总池，不要单行业上限 | 已改为只看国家总工厂上限；例如英国总上限 16，就可以理论上全部发展成 16 个电气化工厂。 |
| 材料按数量购买 | 已改为玩家选择购买数量，各国有不同的每回合购买上限。 |
| 不要两个政府池子 | 已删除“政策专项额度”，只保留政府财政和行政力。 |
| 封锁要让玩家看懂 | 已改为在海外地区上直接显示开放、被封锁或本国独占。 |

## 2. 市场逻辑

### 2.1 国内市场

国内市场现在不是固定价格，也不是硬性限量出售。它会根据本国经济状态自动变化。

玩家可以这样理解：

- 民间购买力越高，国内市场越有钱，同样数量的商品更容易卖出高价。
- 工厂越多、产业越高级，国内社会和产业需求越大，国内市场能承接更多商品。
- 投放商品少于需求时，市场偏缺货，价格更高。
- 投放商品接近需求时，价格接近正常水平。
- 投放商品明显超过需求时，市场偏过剩，价格下降。
- 国内最低成交价不是固定 1，而是“正常单价”的 10%；最高成交价为“正常单价”的 2 倍。

所以前期国内市场通常价格偏低，玩家更适合先靠海外市场拿启动资金。随着收入回流、工厂增加、产业升级，国内市场的购买力和需求会一起成长，后期国内销售会更有价值。

### 2.2 市场计算口径

这里用玩家能理解的方式说明市场价格是怎么推出来的。

国内市场先看两个基础数字：

| 数字 | 含义 | 怎么来 |
| --- | --- | --- |
| 民间购买力 | 本国市场本轮有多少钱买商品。 | 来自上一轮销售收入回流，以及少量初始资源。 |
| 国内承接量 | 本国市场本轮大约能舒服吃下多少商品。 | 来自闲置工厂、手工业、机械化、蒸汽、电气化带来的需求。 |

国内单价的推理顺序为：

1. 先用民间购买力除以国内承接量，得到一个“正常单价”。
2. 再看玩家本轮投放了多少商品。
3. 如果投放少于承接量，说明市场缺货，实际单价会高于正常单价。
4. 如果投放接近承接量，实际单价接近正常单价。
5. 如果投放超过承接量，说明市场过剩，实际单价会低于正常单价。
6. 无论怎么过剩，最低成交价不会低于正常单价的 10%；无论怎么缺货，最高成交价不会超过正常单价的 2 倍。

可以把它理解成：

```text
基础价格 = 民间购买力 ÷ 国内承接量
价格修正因子 = (国内承接量 - 本轮投放量) ÷ 国内承接量
国内实际单价 = 基础价格 × (1 + 价格修正因子)
国内实际单价上限 = 基础价格 × 2
国内实际单价下限 = 基础价格 × 0.1
国内收入 = 四舍五入(国内成交量 × 国内实际单价)
```

其中“国内承接量”只影响价格，不会硬性拦住出售。只要玩家有库存，投放到国内的商品就可以成交；如果投放过多，表现方式是降价，而不是禁止出售。第一回合通过调整各国初始民间购买力，让正常投放时基础价格至少达到 1；后续回合仍完全按消费池和承接量计算，不额外给价格保底。

如果计算出的国内实际单价低于“正常单价 × 0.1”，就按这个下限成交；如果高于“正常单价 × 2”，就按这个上限成交。例如正常单价为 1.05 时，最低成交价就是 0.105，不是固定 0.1 或固定 1。

这个调整系数的含义很简单：

| 投放情况 | 玩家感受 | 价格结果 |
| --- | --- | --- |
| 投放少于承接量 | 市场缺货，买方抢货。 | 单价上升。 |
| 投放约等于承接量 | 市场供需接近平衡。 | 单价接近正常单价。 |
| 投放超过承接量 | 商品倾销，市场吃不下。 | 单价下降，但不低于基础价格的 10%。 |

举例说明：

如果某国民间购买力为 150，国内承接量为 30，那么正常单价就是 5。

| 本轮投放 | 市场状态 | 结果理解 |
| ---: | --- | --- |
| 15 件 | 明显缺货 | 单价会高于 5。 |
| 30 件 | 接近平衡 | 单价约为 5。 |
| 45 件 | 明显过剩 | 单价会低于 5。 |

海外市场的计算更直接：

```text
海外实际成交量 = 不超过库存、不超过海外容量、不被封锁影响后的可卖数量
海外收入 = 海外实际成交量 × 对应区域固定单价
```

例如欧洲固定价为 8。玩家向欧洲投放 6 件商品，但本轮剩余海外容量只有 4，那么实际只能成交 4 件，海外收入为 32。
如果欧洲被其他国家封锁，则本轮对欧洲的可卖数量视为 0。

最后，国内收入和海外收入会合并成国家销售收入，再按 3 : 3 : 4 回流：

```text
民间购买力获得 30%
工厂预算获得 30%
政府财政获得 40%
```

因此市场不是孤立系统。卖货收入会提高下一轮民间购买力，也会给工厂和政府提供预算，从而推动下一轮生产、扩张和研究。

### 2.3 海外市场

海外市场现在使用固定区域价格。

| 海外区域 | 固定单价 | 玩家理解 |
| --- | ---: | --- |
| 欧洲 | 8 | 工业化程度和消费能力最高，是最高价成熟市场。 |
| 美洲 | 7 | 新兴市场购买力较强，价格略低于欧洲。 |
| 亚太 | 6 | 人口和贸易规模大，但不按高价市场处理。 |
| 中东 | 5 | 战略位置重要，但商品购买力低于欧美。 |
| 非洲 | 4 | 当期购买力较弱，价值主要来自容量和战略位置。 |

海外市场的核心不是“卖得更贵”，而是“能卖多少”。

当前海外出售规则：

- 海外区域默认开放，不再需要建交。
- 海外收入 = 成交数量 × 区域固定单价。
- 玩家有一个共享海外容量，用来限制本轮最多能向海外卖多少。
- “贸易促进”提高海外容量。
- 市场竞争成功后，该区域本轮额外可售容量 +2。
- 如果某个区域被其他国家封锁，玩家对该区域的可售容量视为 0。

### 2.4 封锁和独占

封锁现在直接显示在海外地区上，玩家不需要理解底层航线节点。

玩家看到的状态会接近：

- 欧洲：开放。
- 美洲：开放。
- 非洲：被某国封锁。
- 中东：本国独占。
- 亚太：开放。

含义如下：

- 被其他国家封锁的地区，当前不能出售。
- 被本国控制封锁的地区，表示本国在该地区有独占优势，其他国家不能正常出售。
- 封锁的价值体现在“容量和准入”，不是提高商品单价。

## 3. 工厂逻辑

### 3.1 总工厂池

每个国家现在只有一个总工厂池，不再给手工业、机械化、蒸汽、电气化分别设置独立上限。

例如英国总工厂上限为 16，只要科技、材料和预算允许，英国理论上可以全部发展为 16 个电气化工厂。俄罗斯总工厂上限为 20，也可以集中发展为 20 个同类高级工厂。

| 国家 | 总工厂上限 | 单个行业口径 |
| --- | ---: | --- |
| 英国 | 16 | 不单独限制，只受总工厂池限制。 |
| 法国 | 15 | 不单独限制，只受总工厂池限制。 |
| 普鲁士 | 16 | 不单独限制，只受总工厂池限制。 |
| 奥地利 | 14 | 不单独限制，只受总工厂池限制。 |
| 俄罗斯 | 20 | 不单独限制，只受总工厂池限制。 |

闲置工厂代表国家还没有启用的工厂名额。它不生产商品，但会产生少量国内需求，表示人口、城市和产业基础已经存在。

### 3.2 扩建和升级

工厂有两种发展方式：扩建和升级。

| 动作 | 规则 | 玩家理解 |
| --- | --- | --- |
| 扩建 | 直接建设目标类型工厂，本回合立即生效；消耗闲置名额、工厂预算，并受科技前置限制。 | 快速得到想要的工厂，但成本更高。 |
| 升级 | 把已有低级工厂逐级转成高级工厂，本回合立即生效；不增加总工厂数量。 | 依靠已有工业基础逐步提高效率，成本更低。 |

升级路径固定为：

```text
闲置工厂 -> 手工业 -> 机械化 -> 蒸汽 -> 电气化
```

不能跳级升级。例如不能从闲置工厂直接变成机械化，也不能从手工业直接变成蒸汽。

当前成本如下：

| 目标工厂 | 直接扩建成本 | 逐级升级成本 |
| --- | ---: | ---: |
| 手工业 | 13 | 6 |
| 机械化 | 26 | 10 |
| 蒸汽 | 40 | 16 |
| 电气化 | 56 | 24 |

### 3.3 产出和需求

越高级的工厂，材料转化效率越高，同时也会带来更高的国内需求。

| 工厂状态 | 每 1 材料产出商品 | 每 1 工厂带来的国内需求 |
| --- | ---: | ---: |
| 闲置工厂 | 0 | 1 |
| 手工业 | 1 | 2 |
| 机械化 | 2 | 3 |
| 蒸汽 | 4 | 4 |
| 电气化 | 8 | 5 |

这会让游戏形成清楚的成长节奏：前期产出低、国内需求低；后期工厂升级后，产出增加，国内需求也增加，国内市场逐渐变得更有价值。

## 4. 材料购买

材料现在按数量购买，玩家可以清楚选择本回合买多少材料。

当前规则：

- 购买材料消耗工厂预算。
- 当前单位成本为 1 工厂预算购买 1 原材料。
- 购买的材料本回合立即可用于生产。
- 购买材料不会永久提高每回合自然产出。
- 各国每回合材料购买上限不同，用来体现国家资源条件差异。

| 国家 | 每回合材料购买上限 |
| --- | ---: |
| 英国 | 15 |
| 法国 | 16 |
| 普鲁士 | 17 |
| 奥地利 | 15 |
| 俄罗斯 | 20 |

俄罗斯材料上限更高，体现原料供应更充足；英国和奥地利上限较低，体现本土资源和人口条件限制。

## 5. 财政和成长循环

当前玩家主要看到四类资源：

- 民间购买力：影响国内市场价格。
- 工厂预算：用于买材料、投料生产、扩建和升级。
- 政府财政：用于研究、军事、政策和政府相关行动。
- 行政力：用于执行政府政策。

销售收入默认按 3 : 3 : 4 回流：

| 去向 | 比例 | 作用 |
| --- | ---: | --- |
| 民间购买力 | 30% | 提高下回合国内市场承接能力。 |
| 工厂预算 | 30% | 支撑下一轮生产、材料购买、扩建和升级。 |
| 政府财政 | 40% | 支撑研究、军事和政策行动。 |

因此游戏节奏会变成：

- 前期：国内购买力低，主要靠海外固定价拿启动资金。
- 中期：销售收入回流后，工厂扩建和升级，国内需求开始增长。
- 后期：高级工厂产量高，国内市场购买力也更强，玩家需要在国内高价、海外固定价、海外容量和封锁风险之间取舍。

## 6. 本轮版本口径

本轮客户提出的核心逻辑，当前版本已经统一为以下口径：

1. 国内市场按购买力、需求和投放量动态定价。
2. 海外市场按区域固定价出售，欧洲 > 美洲 > 亚太 > 中东 > 非洲。
3. 海外出售不再需要建交。
4. 海外政策只保留“贸易促进”，用于提高海外容量。
5. 市场竞争成功只增加额外可售容量，不提高售价。
6. 封锁直接体现在海外地区上，被封锁地区对其他国家容量为 0。
7. 工厂只受国家总工厂上限限制，不再有单个行业上限。
8. 扩建和升级本回合立即生效；扩建成本高，升级成本低但必须逐级。
9. 材料按数量购买，并按国家设置不同购买上限。
10. 政府财政只保留一个池子，不再显示政策专项额度。
11. 销售收入按 3 : 3 : 4 回流，支撑国内市场、工厂发展和政府行动。
