# 前台文案与术语规范

## 1. 正式术语

前台统一使用以下正式术语：

- `国家收入`
- `国内销售额`
- `海外销售额`
- `国内消费市场`
- `工厂`
- `政府财政`
- `科技点`
- `军事点`
- `国家决策`
- `市场出售`
- `财政结算`
- `累计国家收入`

## 2. 阶段名称

前台阶段名称固定为：

1. `国家决策`
2. `市场出售`
3. `财政结算`

禁止继续出现：

- `生产阶段`
- `军事阶段`
- `政治阶段`

除非是在历史日志中引用旧数据，不允许作为正式界面名称。

## 3. 预算池文案

### 国内消费市场

推荐解释文案：

- 用于扶持国内消费并提升下一轮国内市场能力

### 工厂

推荐解释文案：

- 用于生产、用工、开厂、扩产与工业路线升级

### 政府财政

推荐解释文案：

- 用于购买点数并执行科技、军事与国家政策

## 4. 禁用历史口径

以下历史口径不再作为正式前台文案使用：

- 历史英文三池命名
- 历史单钱包叙事
- 历史结果值口径
- 历史阶段名称

## 5. 容易混淆的表达

必须明确区分：

- `国内销售额`
  - 本回合卖到国内市场赚到的钱
- `国内消费市场`
  - 国家收入分配后的预算池

必须明确区分：

- `国民消费决策`
  - 消耗 `国内消费市场`
- `扶持国内市场政策`
  - 消耗 `政府财政`

## 6. 推荐按钮与标签

推荐按钮：

- `提交国家决策`
- `提交出售方案`
- `查看财政结算`

推荐标签：

- `本回合国家收入`
- `当前收入分配比例`
- `本回合国内销售额`
- `本回合海外销售额`
- `累计国家收入`

## 7. 页面说明文案方向

所有说明文案都应优先回答：

- 这一步花的是哪一个预算池
- 这一步会影响哪一个系统
- 这一步会不会改变国家结构
