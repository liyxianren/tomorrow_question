## 3. 当前项目一句话

当前项目已经具备可运行的网页策略游戏流程和 UI 骨架，但核心经济机制在 2026-04-27 会后被确认需要重构：工业区与市场必须从当前简化 / 随机价格模型，改为基于 `供给 -> 需求 -> 价格 -> 投放策略 -> 国家收入` 的宏观经济模型。

## 4. 当前保留模块

- 首页 / 大厅 / 身份确认
- 房间创建 / 加入 / 选国 / Ready
- AI 补位与自动开局
- 主地图与五个入口 UI
- 议会厅、军事要塞、研究院暂时保持现状
- 刷新恢复、Socket 同步、Bot 自动提交

## 5. 当前优先重构模块

- [第一阶段：市场与生产机制](./第一阶段-市场与生产机制.md)
- 工业区：生产模型、原材料 / 商品抽象、工厂效率、需求创造
- 市民广场 / 市场：供需关系、消费池、动态价格
- 贸易港：本国 / 海外市场投放策略与价格联动
- 财政结算：国家收入来源与三池回流口径
