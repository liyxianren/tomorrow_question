# 用户原始需求：核心机制

> 来源：用户原始需求文档
> 文档定位：2.0 版本核心机制的唯一需求源。
> 使用原则：本文件为最高优先级需求依据。所有后续开发以本文件为准。

---

## 1. 游戏定位

玩家扮演 19 世纪的某一列强国家，带领本国在纷繁复杂的国际局势中走向工业化。

游戏分为**生产、市场、行政、军事**四个阶段。

- 生产阶段：通过投资企业、改良生产方式等操作，将原材料转化为商品。
- 市场阶段：将生产阶段的商品投入到本国市场、国际市场与殖民地，赚取利润并依据分配制度进入消费、投资、财政三个资金池。
- 进入消费池的资金影响本国市场的消费水平。
- 进入投资池的资金可用于投资建设新产能。
- 进入财政池的资金可用于建设科研机构、政府部门与军事机构，进而进行科研开发、政治改革与殖民地扩张。

三者分配比例初始为 **5 : 3 : 2**，后续可根据政策调整。

## 2. 生产与科技

每回合第一阶段为生产阶段。玩家使用投资池扩建产能以及升级落后生产方式，将原材料转化为更多商品。升级产能仅需投入半价。

### 2.1 生产方式

| 生产方式 | 投产比 | 需求 | 产能价格 | 解锁科技 |
|---------|--------|------|---------|---------|
| 空闲产能 | —（不生产） | 1 商品 | — | 无 |
| 手工业生产 | 1 原材料 : 1 商品 | 2 商品 | 20 | 无 |
| 机械生产 | 1 原材料 : 2 商品 | 3 商品 | 40 | 珍妮纺纱机 |
| 蒸汽机生产 | 1 原材料 : 4 商品 | 4 商品 | 60 | 瓦特蒸汽机、车床 |
| 电气化生产 | 1 原材料 : 8 商品 | 5 商品 | 80 | 发电、内燃机 |

每一种生产方式需要研发对应科技解锁。

### 2.2 科研机构与研究进展

研发科技需要使用财政资金建设研究设施，并将其投入至研究中。每一研究机构每回合可提供 1 研究进展。

第一个研究该项科技的国家需要另外进行一次突破——突破机制为每回合掷一次骰子，直到骰子数大于等于研究进度。

### 2.3 科技树

```
莱顿瓶（3） → 伏打电堆（4） → 发电（5）
珍妮纺纱机（3） → 车床（4）
贝塞麦炼钢（3） → 瓦特蒸汽机（4） → 内燃机（5）
```

## 3. 本国市场

每回合第二阶段，玩家需要决策将生产出的商品分别投放多少到本国市场、国际市场与殖民地。

### 3.1 三要素

本国市场机制取决于供给、需求与价格三项：

- **供给**：该回合生产并投入本国市场的商品数量。
- **需求**：取决于本国产能结构，不同生产方式会带来不同需求（见 §2.1 需求列）。
- **价格**：供给与需求的博弈结果。

### 3.2 均衡价格

若供给等于需求，则产品价格为均衡价格：

```
均衡价格 = 本回合消费池资金 / 需求
```

### 3.3 价格波动

- 若供给商品相比于需求商品短缺多少百分比，则价格相较于均衡价格上涨多少百分比。
- 若商品相比于需求商品超过多少百分比，则下降多少百分比。

玩家同时也可决定将生产的一定数量的商品投入到开放的其他玩家的市场、国际市场与殖民地。本国市场和其他玩家所控制市场的开放与关闭取决于本国的政策（详见 §5）。

## 4. 国际市场、殖民地与军事机制

### 4.1 市场独立度等级

国际市场由世界地图上非玩家的区域市场组成，独立度分为四个等级：

| 等级 | 说明 |
|------|------|
| 封闭 | 所有玩家均不能向其销售商品 |
| 开放 | 所有国家均可向其销售商品 |
| 特许 | 只允许特定玩家销售本国商品 |
| 殖民地 | 在特许基础上允许玩家掠夺本区域的产能与原材料 |

### 4.2 区域市场夺取前提

若玩家想夺取某一区域市场，首先要确保本国到达该区域的路线畅通。当本国到达该区域需要通过海洋节点时，还需要投入海军保证航线畅通。

保证每一海洋节点的海上存在需投入最低海军军力。

### 4.3 海上封锁

若某玩家投入某一海洋节点的军力大于其他所有玩家，则该玩家可封锁这一节点，使得其他玩家的商品无法通过该节点流入其他区域市场。

### 4.4 陆军夺取市场

当玩家确保了通往某一区域市场的路线后，还需要投入所需要的陆军军力来打开该区域市场或将该区域夺取成为己方的特许市场/殖民地。

若想要夺取其他玩家的特许市场/殖民地，则需要投入其他玩家在该区域驻扎陆军的 2 倍。

### 4.5 区域独立度变化

每一区域市场的独立度受到市场供需状态与驻扎军力的影响。不同程度的市场短缺与过剩均会导致独立度上升，掠夺产能与原材料也会导致独立度上升，但驻扎更多军力会镇压该地区的独立度。

### 4.6 军力成本

军力需使用财政资金池购买：

| 军力类型 | 成本 |
|---------|------|
| 陆军军力 | 10$ / 点 |
| 海军军力 | 50$ / 点 |

## 5. 政治改革、政策与分配制度

每回合第四阶段为国内政治阶段。玩家将通过改革、施行政策，从而改革分配制度，并维持国内不同意识形态的人群满意度。

### 5.1 三种思潮

每位玩家国内存在三种思潮：自由主义、平等主义、民族主义。

三种思潮的激进度受到市场影响：

| 思潮 | 关联指标 | 变化规则 |
|------|---------|---------|
| 平等主义 | 消费资金池 | 本回合消费资金池上升则激进度下降，反之上升 |
| 自由主义 | 投资资金池 | 与投资资金池挂钩 |
| 民族主义 | 进出口差值 | 进口额大于出口额则下降，出口大于进口则上升 |

### 5.2 行政能力

- 改革需要消耗相应的行政能力。
- 施行政策需要每回合 1 行政能力。
- 行政能力需要财政资金池购买，每 1 行政能力 20$。

### 5.3 常规政策

| 政策 | 效果 |
|------|------|
| 增收消费税 | 减少 5 消费，增加 5% 税收，社会主义激进度 +1 |
| 降低消费税 | 增加 5 消费，减少 5% 税收，社会主义激进度 -1 |
| 扩充军队 | — |
| 缩减军队 | — |
| 扩张科研机构 | — |
| 扩张行政机构 | — |

### 5.4 自由之路改革与解锁政策

| 改革 | 行政力 | 效果 / 解锁政策 |
|------|--------|----------------|
| 制定宪法 | 2 | 各思潮激进度 -1 |
| 股票市场 | 2 | 解锁《公开募股》：本回合可将 10% 消费转化为投资 |
| 建立国会 | 4 | 解锁《民众请愿》：本回合可将 2 点激进度转化为 1 点改革进度 |
| 专利制度 | 4 | 解锁《民间科研》：本回合可使用投资池建设科研机构 |
| 代议制民主国 | 6 | 各思潮激进度 -2 |
| 普选制度 | 6 | 只能使用《民众请愿》推动改革，民族主义激进度 +5 |
| 托拉斯制度 | 6 | 投资比例 +10%，产能价格 -30%，社会主义激进度 +8 |
