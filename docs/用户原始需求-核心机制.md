## 1. 游戏定位

玩家扮演 19 世纪某一列强国家，带领本国在复杂国际局势中完成工业化。

游戏目标不是单纯累积货币，而是通过生产、市场、财政、军事、科研、政治改革与殖民扩张，塑造国家的工业能力、市场结构和国际地位。

## 2. 回合阶段

原始机制中，每回合分为四个阶段：

1. `生产阶段`
2. `市场阶段`
3. `行政 / 国内政治阶段`
4. `军事 / 国际市场阶段`

> 注：原文开头写“生产、市场、行政、军事四个阶段”，后文又写“每回合第四阶段为国内政治阶段”。实际阶段顺序和行政/军事的合并或拆分，需要在重构前再次确认。

## 3. 三个资金池

市场阶段获得的利润，依据分配制度进入三个资金池：

1. `消费池`
   - 影响本国市场消费水平。
   - 是本国市场均衡价格的重要输入。
2. `投资池`
   - 用于投资建设新产能。
   - 用于扩建产能、升级落后生产方式。
3. `财政池`
   - 用于建设科研机构、政府部门、军事机构。
   - 支持科研开发、政治改革、殖民地扩张、军力购买等。

初始分配比例为：

```text
消费 : 投资 : 财政 = 5 : 3 : 2
```

后续可根据政策调整。

## 4. 生产与科技

### 4.1 生产阶段目标

每回合第一阶段为生产阶段。

玩家可以使用 `投资池`：

- 扩建产能
- 升级落后生产方式
- 将原材料转化为商品

规则：

- 升级产能仅需投入半价。
- 不同生产方式有不同投产比、需求、产能价格与科技门槛。

### 4.2 生产方式表

| 生产方式 | 投产比 / 产出 | 需求 | 产能价格 | 科技门槛 |
| --- | --- | --- | --- | --- |
| 空闲产能 | 不生产商品 | 1 商品 | 未给出 | 无 |
| 手工业生产 | 1 原材料 : 1 商品 | 2 商品 | 20 | 无 |
| 机械生产 | 1 原材料 : 2 商品 | 3 商品 | 40 | 珍妮纺纱机 |
| 蒸汽机生产 | 1 原材料 : 4 商品 | 4 商品 | 60 | 瓦特蒸汽机、车床 |
| 电气化生产 | 1 原材料 : 8 商品 | 5 商品 | 80 | 发电、内燃机 |

### 4.3 核心生产逻辑

生产系统的关键不是“多种具体商品”，而是：

```text
原材料 -> 商品
```

不同生产方式体现为：

- 同样原材料能产出多少商品
- 生产方式自身创造多少需求
- 产能建设与升级成本
- 对科技的依赖

因此工厂既改变供给，也改变需求。

### 4.4 科研机构与研究进展

科技需要使用 `财政池` 建设研究设施，并将研究设施投入研究。

规则：

- 每一个研究机构每回合提供 `1` 研究进展。
- 第一个研究某项科技的国家需要额外完成一次“突破”。
- 突破机制：每回合掷一次骰子，直到骰子数大于等于研究进度。

### 4.5 科技树

原文科技树：

```text
莱顿瓶（3） -> 伏打电堆（4） -> 发电（5）
珍妮纺纱机（3） -> 车床（4）
贝塞麦炼钢（3） -> 瓦特蒸汽机（4） -> 内燃机（5）
```

待确认：

- “珍妮纺纱机-（3）-车床(4)”原文格式不完整，整理为珍妮纺纱机（3）-> 车床（4）。
- 科技是否允许并行研究。
- 突破骰子的面数、失败是否累计、多个国家竞争时的处理。

## 5. 本国市场机制

### 5.1 市场阶段目标

每回合第二阶段，玩家需要决策将生产出的商品分别投放到：

- 本国市场
- 国际市场
- 殖民地
- 开放的其他玩家市场

### 5.2 本国市场三要素

本国市场机制由三项决定：

1. `供给`
2. `需求`
3. `价格`

### 5.3 供给

供给为：

```text
本回合生产并投入本国市场的商品数量
```

### 5.4 需求

需求取决于本国产能结构。

不同生产方式会带来不同需求：

- 空闲产能：1 商品需求
- 手工业生产：2 商品需求
- 机械生产：3 商品需求
- 蒸汽机生产：4 商品需求
- 电气化生产：5 商品需求

因此需求不是固定常数，而是由本国工业化结构产生。

### 5.5 均衡价格

若供给等于需求，则商品价格为均衡价格。

公式：

```text
均衡价格 = 本回合消费池资金 / 需求
```

> 注：会议纪要中曾表述为“上回合消费池资金 / 总需求”；原始 docx 写的是“本回合消费池资金除以需求”。实现前需要确认消费池取值时点。

### 5.6 供需导致价格波动

价格围绕均衡价格波动。

规则：

```text
供给 = 需求 -> 价格 = 均衡价格
供给 < 需求 -> 短缺百分比是多少，价格相对均衡价格上涨多少百分比
供给 > 需求 -> 过剩百分比是多少，价格相对均衡价格下降多少百分比
```

可表达为：

```text
短缺率 = (需求 - 供给) / 需求
短缺时价格 = 均衡价格 × (1 + 短缺率)

过剩率 = (供给 - 需求) / 需求
过剩时价格 = 均衡价格 × (1 - 过剩率)
```

待确认：

- 价格是否允许跌到 0 以下，是否需要价格下限。
- 过剩率超过 100% 时如何处理。
- 多市场投放后，本国市场和海外市场价格是否独立计算。

## 6. 国际市场、殖民地与军事机制

### 6.1 市场独立度等级

国际市场由世界地图上非玩家的区域市场组成。

区域市场独立度 / 控制状态分为四个等级：

1. `封闭`
   - 所有玩家均不能销售商品。
2. `开放`
   - 所有国家均可销售商品。
3. `特许`
   - 只允许特定玩家销售本国商品。
4. `殖民地`
   - 在特许基础上，允许玩家掠夺该区域的产能与原材料。

> 原文“寻允许玩家”应为“允许玩家”。

### 6.2 区域市场夺取前提

玩家想夺取某一区域市场，需要先确保本国到该区域的路线畅通。

如果路线需要通过海洋节点：

- 需要投入海军保证航线畅通。
- 每个海洋节点需要最低海军军力维持海上存在。

### 6.3 海上封锁

若某玩家在某一海洋节点投入的军力大于其他所有玩家，则该玩家可以封锁该节点。

效果：

- 其他玩家商品无法通过该节点流入其他区域市场。

待确认：

- “大于其他所有玩家”是大于每个单独玩家，还是大于其他玩家合计。
- 封锁是否影响己方贸易。
- 封锁持续时间和解除条件。

### 6.4 陆军夺取市场

当通往区域市场的路线畅通后，玩家还需要投入陆军军力：

- 打开区域市场
- 将区域市场变为己方特许市场
- 将区域市场变为殖民地

若要夺取其他玩家的特许市场 / 殖民地：

```text
所需陆军 = 对方在该区域驻扎陆军的 2 倍
```

### 6.5 区域独立度变化

每一区域市场的独立度受以下因素影响：

- 市场供需状态
- 驻扎军力
- 产能 / 原材料掠夺

规则：

- 不同程度的市场短缺会导致独立度上升。
- 不同程度的市场过剩也会导致独立度上升。
- 掠夺产能与原材料会导致独立度上升。
- 驻扎更多军力会镇压该地区独立度。

待确认：

- 独立度数值范围。
- 短缺 / 过剩对独立度的具体公式。
- 独立度达到阈值后的事件或惩罚。

### 6.6 陆军与海军成本

军力使用 `财政池` 购买。

| 军力类型 | 成本 |
| --- | --- |
| 陆军军力 | 10$ / 点 |
| 海军军力 | 50$ / 点 |

## 7. 政治改革、政策与分配制度

### 7.1 国内政治阶段

每回合第四阶段为国内政治阶段。

玩家通过：

- 改革
- 施行政策

来改变分配制度，并维持国内不同意识形态人群的满意度。

### 7.2 三种思潮

每位玩家国内存在三种思潮：

1. `自由主义`
2. `平等主义`
3. `民族主义`

激进度变化规则：

| 思潮 | 关联指标 | 变化规则 |
| --- | --- | --- |
| 平等主义 | 消费资金池 | 本回合消费资金池上升，则激进度下降；反之上升 |
| 自由主义 | 投资资金池 | 与投资资金池挂钩，具体涨跌规则待确认 |
| 民族主义 | 进出口差值 | 进口额大于出口额则下降；出口大于进口则上升 |

> 原文政策中多次出现“社会主义激进度”，但前文三种思潮为“自由主义 / 平等主义 / 民族主义”。这里暂按原文保留，并标记需确认“社会主义”是否等同“平等主义”。

### 7.3 行政能力

改革与政策消耗行政能力。

规则：

- 改革需要消耗相应行政能力。
- 施行政策每回合需要 `1` 行政能力。
- 行政能力需要使用 `财政池` 购买。
- 每 `1` 行政能力成本为 `20$`。

## 12. 对当前 Demo 的重构含义

根据本原始需求，当前 Demo 后续不能只保留简化三阶段与随机价格模型。

必须优先恢复 / 重构的核心：

1. `生产阶段`
   - 原材料与商品抽象
   - 生产方式投产比
   - 产能价格和半价升级
   - 科技解锁生产方式
2. `市场阶段`
   - 供给、需求、价格三要素
   - 消费池 / 需求决定均衡价格
   - 供需差决定价格涨跌
   - 本国、国际、殖民地和其他玩家市场投放
3. `国际市场与军事`
   - 区域市场封闭 / 开放 / 特许 / 殖民地等级
   - 海军节点与封锁
   - 陆军打开 / 夺取市场
   - 独立度机制
4. `政治改革`
   - 三种思潮激进度
   - 行政能力
   - 政策与分配制度调整

可暂时保留的现有内容：

- 首页、大厅、房间、选国、Ready、AI 补位
- 主地图 UI 和点击入口
- 静态需求对齐页
- 现有议会厅、军事要塞、研究院 UI 可作为临时壳，但其机制需以后续重构为准
