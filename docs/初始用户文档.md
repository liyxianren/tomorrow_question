# 初始用户文档

## 1. 先理解这局游戏

你在这款游戏里扮演一个 `19` 世纪工业化国家。  
你的目标不是单纯囤钱，而是在 `15` 回合里，让国家持续通过国内和海外市场卖出商品，形成更高的累计国家收入。

每回合都按同一个顺序运行：

`国家决策 -> 市场出售 -> 财政结算`

## 2. 一回合里你要做什么

### 第一步：国家决策

你会同时面对三类决策：

#### 工厂决策

你要决定：

- 这回合生产什么
- 是否支付用工成本开工
- 是否开办新厂
- 是否扩产
- 是否升级工业路线

#### 国民消费决策

你要决定：

- 要不要投入预算扶持国内消费
- 要不要提高下一轮国内市场承接能力
- 要不要让国内市场给出更高收购价格

#### 政府财政决策

你要决定：

- 要不要购买科技点
- 要不要购买军事点
- 要不要研发新路线
- 要不要执行军事扩张
- 要不要执行国家补贴、扶持国内市场等政策

### 第二步：市场出售

你要把本轮产出的商品分到：

- 国内市场
- 海外市场

你还要决定：

- 每类商品卖多少
- 是否愿意承担卖不掉后压仓库的风险

### 第三步：财政结算

系统会帮你算出：

- 本回合国内销售额
- 本回合海外销售额
- 本回合国家收入

然后把国家收入按当前比例，自动分到三个预算池：

1. `国内消费市场`
2. `工厂`
3. `政府财政`

## 3. 三个预算池怎么理解

### 国内消费市场

这部分钱只负责国内消费经营。  
它决定你下一轮的国内市场有没有能力承接更多商品、给更高价格。

### 工厂

这部分钱只负责工业系统。  
你生产、扩产、开厂、升级，都要靠它。

### 政府财政

这部分钱只负责国家层动作。  
科技点、军事点、科技策略、军事策略、部分国家政策，都由它承担。

## 4. 记住两件最容易混淆的事

### 不是所有钱都能随便挪

这三个预算池是独立的，不是一个共享钱包。  
每个池只能干自己的事。

### 国民消费决策不等于政府扶持内需政策

- `国民消费决策` 花的是 `国内消费市场`
- `政府扶持内需政策` 花的是 `政府财政`

它们都可能提升国内市场，但来源不同、层级不同。

## 5. 什么叫国家收入

本版统一使用 `国家收入` 这一概念：

`国家收入 = 本回合国内销售额 + 本回合海外销售额`

这表示国家这一回合真正赚到的钱。

## 6. 为什么国家结构会变化

国家收入初始按 `3 : 3 : 4` 分配到三池。  
但当你执行政府策略后，这个比例会变化。

例如：

- 海外扩张成功后，国家可能更偏向政府财政与海外方向
- 国家补贴政策则可能把结构重新拉回国内消费

所以你玩的不是“固定预算小游戏”，而是不断改变国家结构。

## 7. 怎么赢

整局只比一个主指标：

**15 回合累计国家收入**

谁在 `15` 回合里累计赚到的国家收入最高，谁就赢。
