## 测试原则

- 不随机点按钮；每回合先计算收益和长期价值，再按英国玩家最优路线操作。
- 优先海外高价区域出售；海外容量不足或国内价格更优时再分配国内。
- 工厂预算尽量用于本回合可产出或长期产能增长。
- 政府财政优先用于贸易促进、有效改革、研究/军事中能提高长期收入的动作。
- 非自然最优路线不会为了覆盖按钮强行乱点；未自然覆盖的按钮记录到末尾。
- P0 立即中断测试、记录、修复、推送 GitHub 并回云端复验；P1/P2/P3 仅记录。
