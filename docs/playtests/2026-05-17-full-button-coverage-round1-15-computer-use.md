## 重点结论

- 新建/工厂增加价格覆盖：手工业 3（+1 下回合）、机械化 6（+1 下回合）、蒸汽首次从 0 新建为 40（+2 下回合）、蒸汽已有产能后扩建为 12（+1 下回合）、电气首次新建为 55（+2 下回合）。
- 产业升级价格覆盖：机械化 12（手工业 -> 机械化，立即生效）、蒸汽 20（机械化 -> 蒸汽，立即生效）、电气 30（蒸汽 -> 电气，立即生效）。
- 工厂逻辑主线整体成立：新建/扩建增加下回合产能，产业升级不增加总产能但可立即转为更高倍率产线；R14 -> R15 明确验证蒸汽 +1 生效，R15 明确验证电气升级可当轮生产。
- 最大风险仍是玩家体验层：工厂自动“已决策”会让玩家漏操作，行政力提交校验存在 P1 阻断，研究突破失败反馈不足，市场净调整来源不清。
