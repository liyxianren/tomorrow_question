# 内测可玩版交付标准

## 1. 交付目标

首版内测可玩版的定义，不是功能列表勉强拼起来，而是新版国家收入制可以被真实玩家完整玩完一局。

## 2. 必须满足的产品标准

- `5` 人固定房间可正常开局
- 可完整跑满 `15` 回合
- 回合结构固定为：
  - 国家决策
  - 市场出售
  - 财政结算
- 国家收入、三池、科技点、军事点都可读
- 结算页可以看懂累计国家收入结果

## 3. 必须满足的规则标准

- 国家收入定义明确且实际可结算
- 三个预算池独立存在，且不能自由挪用
- 第一阶段一定同时包含：
  - 工厂决策
  - 国民消费决策
  - 政府财政决策
- 政府策略可改变收入分配比例
- 最终胜负按 `15` 回合累计国家收入

## 4. 必须满足的工程标准

- 服务端状态真相稳定
- 刷新恢复可用
- 超时补交可用
- 大厅、房间、对局、结算链路可跑通
- 本地完整测试套件可执行

## 5. 首版可接受的边界

以下内容可以在内测版中暂时简化：

- 美术与动画精修
- 更深的商品树
- 更复杂的海外市场系统
- 人口 / 税收 / 劳动力
- 多实例部署

但不能简化：

- 三阶段闭环
- 国家收入分账逻辑
- 三池独立预算逻辑
- 最终累计国家收入结算
