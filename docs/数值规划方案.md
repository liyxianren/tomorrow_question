# 数值规划方案

## 1. 数值设计目标

首版数值系统服务于国家收入制，而不是做一套完整宏观经济学模拟器。  
数值设计要解决的核心问题只有三个：

1. 三个预算池如何形成真实取舍
2. 五个国家如何形成不同发展倾向
3. 政府策略如何推动国家结构变化

## 2. 首版固定基线

### 国家收入分配

- 初始分配比例固定为 `3 : 3 : 4`
- 对应：
  - `国内消费市场`
  - `工厂`
  - `政府财政`

### 胜负指标

- 固定为 `15` 回合累计国家收入

### 预算池规则

- 三池独立，不可自由挪用
- 工厂动作只花 `工厂`
- 国民消费动作只花 `国内消费市场`
- 政府动作只花 `政府财政`

## 3. 数值调参轴

### 国内消费市场轴

重点调节：

- 国内市场承接能力增幅
- 国内市场收购价格加成
- 国民消费动作的投入产出比

### 工厂轴

重点调节：

- 工业用工成本
- 新厂成本
- 扩产成本
- 升级工业路线成本
- 产能增长速度

### 政府财政轴

重点调节：

- 科技点购买成本
- 军事点购买成本
- 政策动作成本
- 比例变化幅度

## 4. 政府策略设计原则

政府策略要产生两类效果：

1. 当前回合或下一轮能力变化
2. 国家收入结构变化

示例：

- `扩张海外版图`
  - 需要军事点与科技点
  - 强化海外市场能力
  - 推高 `政府财政` 占比
- `国家补贴政策`
  - 主要消耗政府财政
  - 强化国内消费市场
  - 把比例拉回更偏内需方向

## 5. 五国差异化方向

首版差异主要通过“效率倾向”表达，而不是完全不同规则。

### 英国

- 海外扩张效率更高
- 更适合偏财政与海外市场

### 法国

- 三池转化较均衡
- 更适合中庸策略

### 普鲁士

- 工厂扩张和工业推进效率更高
- 更适合工业路线

### 奥地利

- 政府策略更稳
- 更适合中盘稳态经营

### 俄罗斯

- 中后期累积能力更强
- 更适合后程爆发

## 6. 首版不做的复杂数值

以下系统不进入首版数值主模型：

- 人口
- 税收
- 劳动力细分
- 多层汇率
- 复杂债务

它们可以作为后续版本在国家收入制之上叠加。
