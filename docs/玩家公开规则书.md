# 玩家公开规则书

## 1. 你在玩什么

这是一局 `5` 人固定国家、`15` 回合的工业化列强竞争游戏。  
你需要通过工业建设、消费经营、科技策略、军事扩张和市场出售，让自己的国家在 `15` 回合后获得最高的累计国家收入。

## 2. 回合怎么走

每回合固定分三步：

1. `国家决策`
2. `市场出售`
3. `财政结算`

### 国家决策

这一阶段你要同时处理三类事情：

1. `工厂决策`
   - 安排生产
   - 支付工业用工成本
   - 开办新厂
   - 扩产
   - 升级工业路线
2. `国民消费决策`
   - 扶持国内消费市场
   - 提高下一轮国内市场承接能力
   - 提高国内市场收购价格
3. `政府财政决策`
   - 购买科技点
   - 购买军事点
   - 执行科技策略
   - 执行军事策略
   - 执行扶持国内市场的政策

### 市场出售

这一阶段你要决定：

- 哪些商品卖向国内
- 哪些商品卖向海外
- 每类商品卖多少
- 不想卖或卖不掉的商品是否继续压仓库

### 财政结算

系统会统一结算：

- 本回合国内销售额
- 本回合海外销售额
- 本回合国家收入

然后把国家收入按当前比例自动分到三个预算池中，进入下一回合。

## 3. 三个预算池是什么

每回合财政结算后，国家收入会分到三个**独立预算池**：

1. `国内消费市场`
2. `工厂`
3. `政府财政`

它们不是一笔共享的钱，不能自由挪用。

### 国内消费市场

用来强化本国市场，让下一回合国内更能承接商品，也更愿意出高价。

### 工厂

用来生产、扩产、开办新厂和升级工业路线。

### 政府财政

用来购买科技点、军事点，并执行科技、军事和国家政策动作。

## 4. 国家收入怎么来

本版统一使用 `国家收入` 这一概念。

计算方式：

`国家收入 = 国内销售额 + 海外销售额`

也就是说，国家收入代表你这回合真正靠卖货赚到的钱。

## 5. 比例为什么会变化

每个国家的国家收入，默认按 `3 : 3 : 4` 分到三个预算池：

- 国内消费市场：30%
- 工厂：30%
- 政府财政：40%

但政府财政决策可以改变这个结构。

例如：

- `扩张海外版图`
  - 会消耗军事点和科技点
  - 会让国家更偏向海外扩张
  - 可能把比例从 `3 : 3 : 4` 调整为 `2.8 : 3 : 4.2`

反过来，国家补贴政策、扶持国内市场政策会把结构拉回更偏内需的方向。

## 6. 科技点和军事点怎么用

科技点和军事点不是自动获得的，需要先用 `政府财政` 购买。

- `科技点`
  - 用于研发新产品或更高价值工业路线
- `军事点`
  - 用于开拓海外市场、扩张海外版图、提升海外控制力

## 7. 你要怎么赢

整局固定 `15` 回合。

主胜负条件只有一个：

**15 回合累计国家收入最高者获胜**

所以这游戏比的不是谁最会囤钱，而是谁最会把国家收入滚起来。

## 8. 首版的国家差异

五个国家的差异重点不在“按钮完全不同”，而在于：

- 哪类预算更好用
- 哪类策略更适合该国
- 该国更适合偏内需、偏工业还是偏财政

大方向如下：

- `英国`：偏海外扩张
- `法国`：偏均衡经营
- `普鲁士`：偏工业推进
- `奥地利`：偏稳态军政
- `俄罗斯`：偏中后期积累

## 9. 新玩家建议

- 第一回合不要只顾扩产，先看清自己三个预算池的结构
- 市场出售阶段不要把所有商品都压一个市场
- 政府财政不是纯储备池，它决定你能不能改写国家结构
- 国民消费决策和政府扶持内需政策是两套不同来源的动作，不要混用理解
