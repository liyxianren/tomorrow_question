# 明日之问｜5.27_overchange 最终修改文档

> 建议落盘文件名：`docs/5.27_overchange.md`
> 用途：作为 2026-05-27 最后一版玩法大改的需求确认与实现依据。

## Summary

本版修改聚焦 5 件事：政府改革不再出现无效果节点；游戏从 15 回合压缩到 10 回合；殖民机制恢复到军事地区详情抽屉中；新增规则型 AI 玩家指导；自由、平等、民族三条路线的最终改革都会锁定其他路线。

本文件以最后补充的 5 条为准，不纳入早前提到的“本回合政策延续到下回合”规则，除非后续明确重新加入。

## Key Changes

### 1. 政府改革与路线锁定

所有玩家可见改革必须至少满足一项：有真实结算效果、解锁新政策、或触发路线锁定。配置里不能再出现“当前不会立刻给预算或点数”这类空改革。

重点调整：

| 改革 | 处理口径 |
| --- | --- |
| 社会救济 | 解锁《社会福利》：1 行政力，本回合收入向民间购买力倾斜，平等主义压力 -1。 |
| 工会组织 | 解锁《罢工谈判》：1 行政力，本回合产出减半，平等主义压力 -5。 |
| 苏维埃国 | 作为平等路线强改革：平等主义压力归 0，国内容量 +2，但不承担最终锁路线职责。 |
| 义务教育 | 保留每回合科技点 +1，确认 UI 明确展示其效果。 |
| 全民动员 | 解锁《总动员令》：将当前非 idle 生产产能的 50% 永久转化为等量陆军，平等主义压力 +5。 |
| 军国体制 | 作为民族路线最终改革：行政力上限 +2，自由/平等/民族三类思潮各 -3，解锁《暴力镇压》三项政策，并锁定自由、平等路线。 |
| 暴力镇压 | 拆成三个可选政策：镇压自由主义 / 平等主义 / 民族主义；每次消耗 3 陆军，使目标思潮压力 -4。 |
| 法西斯国 | 作为民族路线强改革：民族主义压力归 0，陆军上限 +3，海外容量 +2，但不承担最终锁路线职责。 |
| 托拉斯制度 | 作为自由路线最终改革：锁定平等、民族路线；同时确保“升级成本 -30%”真实进入工厂升级结算。 |

路线最终锁定表：

| 路线 | 最终改革 | 锁定效果 |
| --- | --- | --- |
| 自由 | 托拉斯制度 | 之后不能选择平等、民族路线改革或其路线政策。 |
| 平等 | 计划经济 | 之后不能选择自由、民族路线改革或其路线政策。 |
| 民族 | 军国体制 | 之后不能选择自由、平等路线改革或其路线政策。 |

已完成的其他路线改革不回滚；锁定只影响之后的选择。终点锁定必须落在每条路线列表里的最后一个改革：自由线 `托拉斯制度`，平等线 `计划经济`，民族线 `军国体制`。

5.27 后续重配平补充：

- 自由线偏市场和工厂效率：专利制度补 `科技点 +1`；普选制度改为需要 `建立国会`，提供 `国内容量 +2`，民族主义压力 +3。
- 平等线偏内需和社会稳定：社会救济补 `国内容量 +1`；八小时工作制改为收入向民间购买力倾斜并降低平等主义压力；工会组织补平等主义压力 -2；苏维埃国补 `国内容量 +2`；集体农庄升级折扣从 -50% 调整到 -25%，并补 `国内容量 +1`；计划经济作为终点，改为 `民间购买力/工厂/政府财政 = 4/1/5`，补 `国内容量 +3`，并清零平等主义压力。
- 民族线偏海外、军力和强制稳定：义务教育显示并结算 `每回合科技点 +1`；国家媒体补民族主义 +1、平等主义 -1；现代海关补 `海外容量 +1`；凯恩斯主义补 `国内容量 +1`；法西斯国补 `陆军上限 +3 / 海外容量 +2 / 民族主义归 0`；全民动员补 `陆军上限 +2`；军国体制作为终点，补 `行政力上限 +2`、三类思潮各 -3、解锁三项镇压政策和路线锁定。

### 2. 游戏回合改为 10 回合

新开局总回合数改为 10。模型默认值、前端文案、终局页、测试 fixture、QA 文档都同步改为 10 回合。事件配置中超过第 10 回合才触发的内容要压缩到第 8-10 回合，不能保留玩家永远看不到的 11-15 回合事件。

已有存档不迁移；只保证新游戏按 10 回合创建和结束。

### 3. 殖民机制恢复

殖民入口放在军事页面的地区详情抽屉里，位置在“地区封锁”模块下面，新增“殖民行动”模块。

规则口径：

- 本版把“3 点军事点数”落到现有资源体系，表现为消耗 3 陆军，不新增旧版 `military_points` 资源。
- 只有 `colonizable=true` 的海外地区可以殖民；欧洲不可殖民。
- 地区必须未被任何国家殖民，且当前对玩家可进入。
- 点击“殖民该地区”后进入本回合军事草案；提交后消耗 3 陆军，并永久设置该地区 controller。
- 殖民不改变海外固定价格，不自动形成封锁，不恢复旧征服/掠夺/独立度链条。
- 每回合结算末尾，殖民地返还原材料到下一回合可用原材料。

原材料返还用地区资源上限推导：

```text
殖民地每回合原材料 = max(1, floor(地区 resourceLimit 总和 / 4))
```

当前数值：

| 地区 | resourceLimit 总和 | 每回合原材料 |
| --- | ---: | ---: |
| 美洲 | 13 | +3 |
| 非洲 | 8 | +2 |
| 中东 | 10 | +2 |
| 亚太 | 9 | +2 |
| 欧洲 | 12 | 不可殖民 |

UI 要展示殖民成本、预计每回合原材料、当前控制国、不可殖民原因。

### 4. AI 玩家指导

本版先做规则型 AI 指导，不接外部 LLM，不自动提交。

表现位置：游戏右侧辅助栏新增“AI 指导”模块，按当前阶段和当前资源给出 3-5 条建议。建议包含动作、原因、预期收益或风险。

指导逻辑：

- 工厂：优先避免原材料闲置，提示升级、扩建、购买材料和加班取舍。
- 市场：优先推荐高价海外容量，再处理国内投放；说明封锁和容量限制。
- 政府：推荐有真实效果或关键解锁的改革；点最终改革前提示路线锁定。
- 军事：如果陆军不少于 3，优先推荐殖民原材料收益最高且可进入的地区。
- 研究：推荐当前能立即形成生产或市场收益的科技。
- 终局前：第 8-10 回合建议更偏向现金回流和可立即兑现的动作。

AI 指导只解释和建议，不替玩家点击，不隐藏玩家原有选择。

### 5. 接口与状态变化

需要新增或补齐的公共数据：

- `militaryPlan.colonizationActions: [{ regionId }]`
- 地区状态增加：是否可殖民、殖民成本、预计原材料收益、当前控制国、不可殖民原因。
- 政府改革数据增加：终点改革标记、锁定路线、锁定说明。
- 常规政策引擎支持非 ratio 效果，例如产出倍率、思潮变化、陆军消耗、产能转军力。
- 工作区增加 AI 指导列表，前端只展示，不负责重新推导核心规则。

## Test Plan

后端测试：

- 所有改革都通过“有效果 / 有解锁 / 有终点锁”校验。
- 托拉斯制度、计划经济、军国体制分别触发路线锁定。
- 被锁路线的改革和政策不能再激活。
- 社会福利、罢工谈判、总动员令、暴力镇压按规则结算。
- 新游戏第 10 回合结束，第 11 回合不会出现。
- 殖民消耗 3 陆军，不能殖民欧洲或已殖民地区，结算后按公式返还原材料。

前端测试：

- 军事地区抽屉中，“殖民行动”位于“地区封锁”下面。
- 殖民按钮、成本、收益、禁用原因展示正确。
- 政府改革卡展示解锁、数值和最终锁定提示。
- AI 指导在不同阶段显示合理建议。
- 所有活跃 UI 文案从 15 回合改为 10 回合。

真实浏览器验收：

- 新开一局跑通 1-10 回合并进入终局。
- 在军事页打开地区详情，确认殖民模块位置正确。
- 使用 3 陆军殖民一个可殖民地区，下一回合看到原材料增加。
- 点托拉斯制度 / 计划经济 / 军国体制之一后，其他路线不可再点。
