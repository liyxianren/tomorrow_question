## 问题详情

### P0-1: 殖民解锁门槛过高

**现象**: 殖民解锁需要10军事点。每回合最多3次recruit_infantry(+3mp) + 购买2mp = 5mp/回合。至少需要2回合纯积累军事点，期间完全放弃其他发展。

**真实玩家视角**: 作为Russia玩家，我连续5回合只做军事行动，军事点才积累到6点（其中2回合买了mp）。殖民解锁遥遥无期，游戏体验极差。

**建议修复**: 
- 殖民解锁cost从10降到5-6
- 或者让recruit_infantry每回合+2mp（而非+1）
- 或者增加更多mp获取渠道

### P1-1: recruit_infantry不创建infantry单位

**现象**: `recruit_infantry`的effect只有`militaryPointsDelta: 1`，不改变`army.infantry`数量。army.infantry初始为1，5回合后仍为1。

**游戏性问题**: 玩家会认为"征募步兵"应该增加步兵数量，但实际上只增加了一个抽象的"军事点"。命名误导。

**建议修复**:
- 要么让recruit_infantry同时增加army.infantry
- 要么改名为"征召预备役"或"军事训练"等不暗示具体兵种的名称

### P1-2: 军事点与army单位概念混淆

**现象**: 
- `militaryPoints`: 用于解锁殖民、购买政策的通用货币
- `army.infantry/artillery`: 独立的战斗单位，目前似乎只用于区域驻军

两者的关系不清晰。玩家不清楚"军事点"和"军队"的区别。

**建议**: 在UI中明确区分：
- 军事点 → "军事预算"或"军费"
- army → "作战单位"或"驻军"

## 结论

军事殖民系统的核心问题是**节奏太慢**。从零开始到殖民获利，至少需要：
- 2回合积累mp解锁殖民
- 1回合建交
- 1回合殖民
- 至少2回合回收成本（殖民收入5/回合，解锁+建交+殖民总成本约16+预算）

**7-8回合才能开始从殖民获利**，而游戏总共只有15回合。殖民策略完全不可行。
