# 经济模型修复计划

> 基于 cdh 深度分析 + 当前代码审计
> 目标：让 15 回合游戏有真实的策略深度和价格波动

---

## 一、修改清单

### 1.1 消费池消耗率（核心修复）

**文件**：`backend/app/modules/rules/settlement.py`

在 `_allocate_income_phase1` 分账之后，增加消费池消耗：

```python
# 分账后
consumption = player_state.budget_pools.get("domesticMarket", 0)
drain_rate = Decimal("0.3")  # 30% 消耗率
player_state.budget_pools["domesticMarket"] = int(Decimal(str(consumption)) * (1 - drain_rate))
```

**效果**：
- 消费池自然收敛到 ~500（增量 ≈ 消耗）
- 均衡价稳定在 10~15
- 打断"收入→消费池→价格→收入"的正反馈链

### 1.2 空闲产能需求系数改为 0

**文件**：`backend/app/modules/rules/phase1_economy.py`

```python
PRODUCTION_MODE_DEMAND_COEFFICIENTS: Mapping[str, Decimal] = {
    "idle": Decimal("0"),      # 改：空闲产能不创造需求
    "handicraft": Decimal("1"), # 改：从 2 降到 1
    "mechanized": Decimal("2"), # 改：从 3 降到 2
    "steam": Decimal("3"),      # 改：从 4 降到 3
    "electrified": Decimal("4"),# 改：从 5 降到 4
}
```

**效果**：
- 不再"永远短缺"——空闲产能不人为制造需求
- 需求更温和，价格增长更平缓
- 供给 > 需求时出现真正的过剩 → 海外市场有意义

### 1.3 提高原材料供给

**文件**：`backend/config/balance/countries.json`

所有 5 国 `rawMaterialsPerTurn` 从 5 提高到 **12**。

**文件**：`backend/config/balance/global.json`

`rawMaterialsPerTurn` 从 20 改为 **12**（默认值）。

**效果**：
- 前期有足够原材料分配到不同生产方式
- 后期产能扩张后原材料成为真正瓶颈
- "分配原材料到哪种生产方式"成为有意义的决策

### 1.4 提高初始产能

**文件**：`backend/config/balance/countries.json`

所有 5 国 `productionCapacity.handicraft` 从 4 提高到 **8**。

**效果**：
- 第一回合就有 8 手工业 × 1:1 = 8 商品可分配
- 需求 = 8×1 = 8，均衡价 = 消费池/8，有真实的供需博弈

### 1.5 降低海外市场门槛

**文件**：`backend/config/balance/global.json`

`baseOverseasCapacity` 从 2 提高到 **5**。

**效果**：
- 回合 3~4 就能投海外（不再要等到回合 8+）
- 海外市场成为早期策略选项

### 1.6 调整初始三池比例

**文件**：`backend/config/balance/countries.json`

按用户原始需求，将收入分配比例改为 **5:3:2**（消费:投资:财政）。

当前英国是 2.5:3:4.5，需要改为 5:3:2。

**效果**：
- 符合用户原始需求
- 消费池占比更高 → 价格更高 → 但有 30% 消耗率平衡

## 二、推演验证（消耗率 30% + 新参数）

### 起手状态（英国）

```
消费池: 12 | 投资池: 14 | 财政池: 22
产能: 手工业 8 + 空闲 0
原材料: 25 + 12/回合
需求: 8×1 = 8
均衡价: 12/8 = 1.5
```

### 回合 1

**生产**：投 8 原料 → 产 8 商品。建 1 手工业（-12 投资）。

**市场**：投本国 8，需求 8 → 均衡 1.5，无短缺 → 售 12

**分账**：消费 +6 / 投资 +3.6 / 财政 +2.4

**消耗**：消费池 = (12+6) × 0.7 = **12.6**

```
末状态: 消费 12.6 | 投资 5.6 | 财政 24.4
        产能: 手工业 9 | 原材料: 29
```

### 回合 2

**需求** = 9×1 = **9** | **均衡价** = 12.6/9 = **1.4**

**生产**：投 9 原料 → 产 9。建 1 手工业（-12）。

**市场**：投本国 9，需求 9 → 均衡 1.4 → 售 12.6

**分账**：+6.3 / +3.8 / +2.5

**消耗**：消费 = (12.6+6.3) × 0.7 = **13.2**

```
末状态: 消费 13.2 | 投资 -2.6* | 财政 26.9
        产能: 手工业 10 | 原材料: 33
        *投资不够建新产能 → 玩家需要选择：等一回合 or 升级
```

### 回合 3（关键转折）

**需求** = 10×1 = **10** | **均衡价** = 13.2/10 = **1.32**

**选择 A**：继续建手工业（-12，但投资池负数，等一回合）
**选择 B**：升级手工业→机械（-12，半价），下回合需求变 9×1 + 1×2 = 11，但产出变 9+2=11

**玩家第一次有真正的策略选择！**

### 回合 5+（中期）

```
产能: 手工业 6 + 机械 4 = 10
需求: 6×1 + 4×2 = 14
原材料: 12/回合，能吃 6+4=10，剩余 2
产出: 6 + 8 = 14
均衡价: ~2.5（消费池稳定在 35 左右）
```

### 回合 8+（过剩临界）

```
产能: 手工业 4 + 机械 6 = 10
需求: 4×1 + 6×2 = 16
原材料: 12/回合，能吃 4+6=10，产出 4+12=16
供给 16 = 需求 16 → 均衡
```

**但如果升级到蒸汽机**：
```
产能: 手工业 2 + 机械 4 + 蒸汽 2 = 8
需求: 2×1 + 4×2 + 2×3 = 16
原材料: 12，能吃 2+4+4=10，产出 2+8+8=18
供给 18 > 需求 16 → 过剩 12.5% → 价格下降
→ 可以投海外！
```

## 六、风险

| 风险 | 影响 | 应对 |
|------|------|------|
| 消耗率太高导致消费池枯竭 | 后期无力 | 先用 30%，观察再调 |
| 需求系数改小导致永远过剩 | 海外市场太快 | 可降到 0.5 步进 |
| 原材料太多导致无瓶颈感 | 失去资源管理深度 | 可降到 8~10 |
| 测试数值全部要改 | 工作量大 | 批量 sed 替换 |
