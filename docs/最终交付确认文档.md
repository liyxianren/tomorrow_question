# 19世纪工业化列强策略游戏
# 最终交付确认文档

> 文档定位：首版正式规则总纲  
> 当前版本：2.0

## 1. 产品目标

本项目首版要交付的是一款 `5` 人固定国家、`15` 回合同步提交、统一结算的网页策略游戏。  
玩家扮演 `英国 / 法国 / 普鲁士 / 奥地利 / 俄罗斯` 之一，在有限回合内通过工业建设、国内消费经营、科技与军事策略、海外市场拓展来提升国家收入。

正式回合主循环固定为：

`国家决策 -> 市场出售 -> 财政结算`

## 2. 核心经济系统

### 2.1 国家收入

本版统一使用 `国家收入` 作为核心经济口径。

定义如下：

**国家收入 = 本回合国内销售额 + 本回合海外销售额**

### 2.2 国家收入分配

每回合财政结算后，国家收入自动按当前比例分配到三个**独立预算池**：

1. `国内消费市场`
2. `工厂`
3. `政府财政`

首版默认初始分配比例固定为：

`3 : 3 : 4`

即：

- 国内消费市场：30%
- 工厂：30%
- 政府财政：40%

这三个预算池是**独立消费池**：

- 每个池只负责自己的决策类型
- 玩家不能把一个池里的钱自由挪到另一个池里

### 2.3 三个预算池的职责

#### 国内消费市场

用于提升国家内部消费能力，主要作用是：

- 扶持国内消费市场
- 提高下一轮国内市场承接能力
- 提高国内市场收购价格

#### 工厂

用于工业生产与工业扩张，主要作用是：

- 安排生产
- 支付工业用工成本
- 开办新厂
- 扩产
- 升级工业路线

#### 政府财政

用于国家层的科技、军事与政策动作，主要作用是：

- 购买科技点
- 购买军事点
- 执行科技策略
- 执行军事策略
- 执行扶持国内市场的政策

## 3. 每回合流程

### 3.1 第一阶段：国家决策

玩家在这一阶段完成本回合的国家经营决策。  
首版固定包含三类主动决策：

1. `工厂决策`
2. `国民消费决策`
3. `政府财政决策`

#### 工厂决策

工厂决策消耗 `工厂` 预算池，包括：

- 安排生产
- 支付工业用工成本
- 开办新厂
- 扩产
- 升级工业路线

#### 国民消费决策

国民消费决策消耗 `国内消费市场` 预算池，包括：

- 扶持国内消费市场
- 提高下一轮国内消费市场能力
- 让国内市场给出更高收购价格

#### 政府财政决策

政府财政决策消耗 `政府财政` 预算池，包括：

- 购买科技点
- 购买军事点
- 执行科技策略
- 执行军事策略
- 执行扶持国内市场的政策

这里必须明确：

- `国民消费决策` 不等于 `政府财政中的扶持国内市场政策`
- 前者花 `国内消费市场`
- 后者花 `政府财政`

### 3.2 第二阶段：市场出售

玩家在这一阶段执行本回合的出售安排。

本阶段的核心动作固定为：

- 把本轮产出分配到国内或海外市场
- 决定各类商品卖向哪里
- 决定卖量
- 承担卖不出去后是否压仓库的风险

本阶段的结果是形成：

- 本回合国内销售额
- 本回合海外销售额
- 本回合国家收入

### 3.3 第三阶段：财政结算

系统在这一阶段统一完成：

- 计算本回合国内销售额
- 计算本回合海外销售额
- 汇总为本回合国家收入
- 按当前比例重新分配到三个预算池
- 刷新下一回合状态

## 4. 政府策略与国家结构变化

政府财政动作不只带来即时效果，还会改变国家后续回合的收入结构。

示例：

- 玩家执行 `扩张海外版图`
- 消耗 `3` 个军事点数 + `1` 个科技点数
- 点数本身需要先用政府财政购买

执行后，国家收入分配比例可从：

`3 : 3 : 4`

调整为：

`2.8 : 3 : 4.2`

这表示：

- 国内消费市场占比下降
- 工厂占比保持稳定
- 政府财政占比提升

反过来，国家补贴政策、扶持国内市场政策、刺激内需政策可以把国家结构逐步拉回更偏内需的方向。

比例变化属于**国家结构变化**，不是一次性奖励。  
它会影响后续回合的预算分配方向。

## 5. 科技点与军事点

本版采用“先购买点数，再消耗点数执行动作”的设计：

- `科技点`
  - 用于研发新产品、新工艺和更高价值商品路线
- `军事点`
  - 用于开拓海外市场、扩张海外版图、提升海外控制力

示例：

- `石油炼制`
  - 属于科技方向
  - 可提高高阶商品出售价格
- `扩张海外版图`
  - 属于军事方向
  - 可提高海外市场能力并调整国家收入结构

## 6. 国家差异化方向

首版国家差异不再只体现在开局静态数值，而要主要体现为：

- 工厂效率差异
- 国内消费市场效率差异
- 政府财政转化效率差异
- 科技与军事策略的适配差异
- 哪类收入结构更适合该国

建议的国家倾向如下：

- `英国`：更适合海外扩张与财政强化
- `法国`：更适合均衡经营
- `普鲁士`：更适合工业推进
- `奥地利`：更适合稳态军政
- `俄罗斯`：更适合中后期积累与爆发

## 7. 交付边界

首版必须交付：

- 大厅、房间、准备、开局、对局、结算
- `15` 回合完整闭环
- 三阶段同步提交与统一结算
- 关键数值展示、阶段摘要、排行榜、日志、刷新恢复

首版不做：

- 账号体系
- 匹配、观战、历史战绩
- AI 补位之外的复杂 PVE
- 多实例扩容
- 人口、税收、劳动力等更深层宏观经济系统

## 8. 胜负判定

整局固定为 `15` 回合。

主胜负条件固定为：

**15 回合累计国家收入最高者获胜**

这意味着：

- 不比谁最后囤的钱最多
- 不比谁某一回合爆发最高
- 比的是整局累计创造了多少国家收入

## 9. 本文件结论

本项目首版的核心机制是：

**先形成国家收入，再按国家结构分账，最后通过工厂决策、国民消费决策与政府财政决策塑造下一回合的国家结构。**
