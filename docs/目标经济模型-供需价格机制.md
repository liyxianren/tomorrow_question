# 目标经济模型：供给 - 需求 - 价格机制

> 文档定位：2026-04-27 会议后形成的目标经济模型草案。  
> 当前依据：[用户原始需求：核心机制](./用户原始需求-核心机制.md)。  
> 状态：已根据用户原始文档补齐第一版结构，仍需确认若干公式边界。  
> 适用范围：工业区生产、市民广场 / 市场机制、市场出售与财政结算的核心经济链路。

## 1. 模型目标

目标不是做随机价格游戏，而是做一个可解释的宏观经济模型：

```text
产能结构 -> 供给与需求 -> 价格 -> 销售收入 -> 下一轮消费池 / 预算池
```

玩家决策应围绕以下问题展开：

- 建什么厂？
- 产多少商品？
- 当前市场需求能否承接？
- 供需差会如何改变价格？
- 商品应投放本国市场还是海外市场？

## 2. 核心抽象

### 2.1 原材料

`原材料` 是生产输入。

当前目标模型不以钢铁、茶叶等多种具体商品作为核心，而是先抽象为统一生产输入。

### 2.2 商品

`商品` 是生产输出，也是市场销售对象。

不同工厂的差异不通过多商品种类体现，而通过以下参数体现：

- 原材料消耗
- 商品产出效率
- 需求创造能力
- 工厂等级 / 工厂类型

### 2.3 工厂 / 产能结构

工厂同时影响两件事：

1. 供给：能生产多少商品
2. 需求：会创造多少商品需求

这意味着“建厂”不是单纯增加产出，也会改变国内经济结构和市场容量。

## 3. 供给计算

供给由原材料数量和生产效率决定。

基础形式：

```text
商品供给 = 原材料数量 × 生产效率系数
```

示例：

```text
机械厂效率 = 1:2
投入 1 个原材料
产出 2 个商品
```

待确认项：

- 原材料初始数量
- 每类工厂的效率系数
- 原材料不足时的排产规则
- 产能闲置是否有惩罚或机会成本

## 4. 需求计算

需求由产能结构决定。

基础原则：

- 空闲 / 未建厂状态也存在基础需求，模拟人口基本消费。
- 工厂等级越高，可能同时提高产出和需求。
- 不同工厂的产出效率和需求创造系数可以不同。

示例：

```text
空闲产能 20 -> 基础需求 20
机械厂产出 2 个商品 -> 创造 3 个商品需求
```

待确认项：

- 各类工厂的需求创造系数
- 基础需求是否随回合、国家、人口或政策变化
- 需求是否分本国 / 海外 / 殖民地分别计算

## 5. 价格形成

### 5.1 均衡价格

均衡价格由上回合消费池资金和总需求决定。

```text
均衡价格 = 上回合消费池资金 / 总需求
```

示例：

```text
上回合消费池资金 = 720
总需求 = 72
均衡价格 = 10
```

### 5.2 供需波动

价格围绕均衡价格，按供需差波动。

已确认方向：

```text
供给 < 需求 -> 价格上涨
供给 > 需求 -> 价格下跌
供给 = 需求 -> 接近均衡价格
```

会议示例：

```text
供给 = 64
需求 = 72
缺口 = 8
缺口比例 = 8 / 72 = 1/9
价格相对均衡价格上涨 1/9
```

候选表达：

```text
价格 = 均衡价格 × (1 + (需求 - 供给) / 需求)
```

> 该公式已与用户原始需求中的方向一致；正式实现前仍需确认价格上下限、四舍五入和消费池取值时点。

## 6. 市场投放

玩家需要把商品投放到不同市场。

当前目标市场类型包括：

- 本国市场
- 海外市场
- 殖民地 / 其他玩家市场（待原始文档确认）

策略逻辑：

- 本国供不应求：价格上涨，投放本国可能更赚钱。
- 本国供过于求：价格下跌，过剩商品应考虑海外市场。
- 海外市场应承担“转移过剩产能、维持利润”的功能。

待确认项：

- 海外市场需求如何计算
- 海外市场价格是否独立
- 其他玩家市场是否允许直接投放
- 殖民地市场与军事 / 外交模块如何关联
