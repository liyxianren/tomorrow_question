# 19世纪工业化列强策略游戏

## 项目定位

本项目是一款 `5` 人固定国家、`15` 回合、同步提交、统一结算的网页回合制策略游戏。  
当前仓库的正式规则基线，统一采用**国家收入制**：

`国家决策 -> 市场出售 -> 财政结算`

项目首版的核心目标不是做一套无限扩展的经济模拟器，而是先完成一个可完整跑完 `15` 回合、国家差异清晰、收入结构可被玩家主动塑造的对局闭环。

## 核心规则总览

### 国家收入

- `国家收入 = 本回合国内销售额 + 本回合海外销售额`
- 财政结算后，国家收入按当前比例自动分配到三个**独立预算池**
- 初始分配比例固定为 `3 : 3 : 4`

### 三个预算池

1. `国内消费市场`
   - 用于扶持国内消费
   - 影响下一回合国内市场承接能力与收购价格
2. `工厂`
   - 用于安排生产、支付工业用工成本、开办新厂、扩产、升级工业路线
3. `政府财政`
   - 用于购买科技点、购买军事点、执行科技策略、执行军事策略、执行扶持国内市场的政策

### 每回合三阶段

1. `国家决策`
   - 工厂决策
   - 国民消费决策
   - 政府财政决策
2. `市场出售`
   - 决定卖向
   - 决定卖量
   - 承担压仓库风险
3. `财政结算`
   - 计算国内销售额
   - 计算海外销售额
   - 汇总国家收入
   - 按比例重新分配到三池

### 胜负判定

- 整局固定 `15` 回合
- 主胜负条件固定为：`15` 回合累计国家收入最高者获胜
